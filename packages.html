QuantumLite Package

QuantumLite offers a simple, yet useful discord bot, which includes the following commands.:

  • !help - A list of bot commands.
  • !info - Provides basic info about the bot (version, creator, etc.).
  • !ping - Displays the bot's current latency.
  • !roll - Rolls a six-sided dice and displays the result.
  • !quote - Fetches a random inspirational or motivational quote.
  • !joke - Tells a random joke.
  • !fact - Shares an interesting fact.
  • !time [timezone] - Displays the current time for the given timezone.
  • !calc [expression] - Performs basic arithmetic calculations based on the provided expression.
  • !wiki - Searches Wikipedia for information based on the user's query and provides a summary.

QuantumPro Package

The QuantumPro Package is a bit more advanced than the Lite one, and offers more commands, and better customization.

  • Advanced Search Command: !search <query>
  • Custom Commands: !custom <command> <response>
  • Auto-Role Assignment: !autorole <role>
  • Scheduled Messages: !schedule <time> <message>
  • Advanced Polls: !poll <question> <options>
  • Welcome Message Customization: !welcome <message>
  • Meme Generation: !meme <template> <text>
  • Custom Reactions: !reaction <trigger> <response>
  • Nickname Management: !nickname <user> <nickname>
  • Moderation Tools: !kick, !ban, !mute, !warn, etc.
  • Voice Channel Management: !move <user> <channel>, !mutevc <user>, !unmutevc <user>, etc.
  • Advanced User Info: !userinfo <user>
  • Role Management: !createrole, !assignrole <user> <role>, !removerole <user> <role>, etc.

QuantumPrime Package

The QuantumPrime package, offers a wide-variety of commands from the start, but this package can be customized depending on how you see fit!

  • Bot Customization: !custombot <settings>
  • Custom Commands Creation: !createcommand <command> <response>
  • Advanced Auto-Role: !autoroleplus <settings>
  • Scheduled Announcements: !announce <time> <message>
  • Complex Polls: !advancedpoll <question> <options> <settings>
  • Custom Welcome DM: !welcomedm <message>
  • Image Generation: !generate <template> <parameters>
  • Custom Reactions: !reactionplus <settings>
  • User Profile Personalization: !profile <settings>
  • Advanced Moderation: !auditlog, !clean, !warnsettings, etc.
  • Voice Channel Automation: !voiceautomation <settings>
  • Custom User Info: !customuserinfo <settings>
  • Role Management Plus: !rolemanagementplus <settings>
  • Custom API Integration: !apiintegration <settings>
